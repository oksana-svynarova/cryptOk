# CryptOk

A modern and user-friendly cryptocurrency tracking application built with React to provide real-time market insights and detailed analytics.

## Features
- **Coins List**: A full list of cryptocurrencies, sorted by market capitalization by default.
- **Advanced Sorting**: Sort coins by name, price, 24h change, trading volume, or market cap in both ascending and descending order.
- **Search Functionality**: Quickly find any coin using the search bar at the top of the list.
- **Load More**: Dynamically load additional coins with the “Load more” button.
- **Watchlist Support**: Add or remove coins from your personal watchlist by toggling the heart icon. All sorting options are available in the watchlist as well.
- **Detailed Coin Page**: View essential data such as price, rank, market cap, volume, supply, and all-time-high, along with an interactive chart featuring multiple time ranges (7–365 days).
- **Multiple Navigation Paths**: Open coin pages directly from the list or via the global header search.
- **Theme Toggle**: Switch between dark and light themes, with dark mode enabled by default.

## Technologies
- **JavaScript** – Core programming language of the application.
- **React** – UI library.
- **Vite** – Fast development and optimized production builds.
- **Axios** – For handling API requests.
- **GitHub Pages** – For hosting the live demo.
